Conexão com Google Sheets

Com a URL configurada, os dados são salvos e lidos diretamente do Google Sheets. Qualquer computador que abrir este arquivo vai carregar os dados atualizados automaticamente.

Atualizar código do Apps Script

Se você já configurou antes, substitua o código antigo por este — ele agora também consegue devolver os dados para o sistema.

Clique em "Copiar código" abaixo.
Backup manual

Como configurar — passo a passo
1

Crie uma planilha no Google Sheets

Acesse sheets.google.com, crie uma planilha nova e dê o nome "Bordado que Amo — Presença".

2

Abra o Apps Script

Na planilha clique em Extensões → Apps Script. Apague o código existente e cole o código copiado acima.

3

Publique como aplicativo web

Clique em Implantar → Nova implantação. Tipo: "Aplicativo da Web". Execute como: "Eu". Acesso: "Qualquer pessoa". Clique em Implantar e autorize.

4

Cole a URL acima e clique em Salvar

Pronto! O sistema vai carregar e salvar os dados automaticamente em qualquer computador.
